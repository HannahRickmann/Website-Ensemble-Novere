# 🎶 Website des Frauenensembles Novere

Dies ist der veröffentlichte Code für die offizielle Website unseres selbst gegründeten Frauenensembles.  
Die Website dient als zentrale Anlaufstelle für Konzertankündigungen, Informationen über das Ensemble sowie Kontaktmöglichkeiten zum Verein.


## ✨ Inhalt & Zweck der Website

Die Website bietet:

- **Aktuelle Konzertankündigungen**  
  Termine, Programme und Veranstaltungsorte unserer Auftritte.

- **Über uns**  
  Hintergrund zum Ensemble, musikalische Ausrichtung, Mitgliederinformationen.

- **Vereins- & Kontaktinformationen**  
  Kontakt zum Ensemble/Verein, Buchungsanfragen, organisatorische Details.
